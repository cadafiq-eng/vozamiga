# VozAmiga 🔊

App de comunicación aumentativa para personas con baja capacidad auditiva.

**URL:** https://cadafiq-eng.github.io/vozamiga/

## Funciones
- ⌨️ Escritura con texto a voz (TTS) en español
- 🎙️ Modo amplificador de micrófono en tiempo real
- 💬 Librería de frases rápidas por categoría
- 📋 Historial de mensajes
- 👩/👨 Voz femenina y masculina alternables
- 📱 Instalable como app (PWA)

## Instalación en Android
1. Abrir https://cadafiq-eng.github.io/vozamiga/ en Chrome
2. Menú ⋮ → "Agregar a pantalla de inicio"
3. Confirmar — aparece como app independiente

## Tecnología
HTML5 · Web Speech API (TTS) · Web Audio API · Service Worker (PWA)
